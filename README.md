## Description

Sample personal application

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

```

and visit http://localhost:3000/api/v1/available-products/1111


```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
